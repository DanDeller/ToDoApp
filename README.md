# ToDo
A quick todo app with node, express, angularJS, and rethinkdb

<h2>Start up instructions:</h2>

<h3>Set up app:</h3>
```
cd seoApp
npm install
```

<h3>Start rethinkdb:</h3>
```
rethihkdb
```

<h3>Start up app:</h3>
```
npm run dev
```

<h3>Start up gulp:</h3>
```
cd public
gulp
```

<h2>View app and rethink data:</h2>

<h3>Front end</h3>
```
http://localhost:3000/#/
```

<h3>RethinkDB data explorer</h3>
```
http://localhost:8080/#dataexplorer
```
